Each particle type has an integer mass (1–8). Mass attenuates applied force (inertia). Transforms must conserve mass: a mass-4 particle can split into 2×mass-2 or 4×mass-1; to form a higher-mass type it consumes same-type neighbors within range.

Mass per type

Each tick a particle accumulates force from each trigger type. If it exceeds the threshold the particle probabilistically transforms — the further past the threshold, the higher the chance (up to 50%/tick at 3× threshold). First match wins.

Row = source type · Col = trigger type

Polar field per type

M = monopole (uniform). 2 = dipole (front+/back−, velocity-aligned). 3–6 = N lobes evenly spaced; click ± to flip each lobe's polarity. Positive lobes amplify the base force, negative lobes reverse it.
